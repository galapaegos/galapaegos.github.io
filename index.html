Here are some examples of software I have helped developed.

Virtual Temporal Bone Surgery Simulator

Perform virtual ear surgery using 3D force-feedback devices. Subjects remove bone using virtual representations of surgical instruments, see volume removed instantly, and flush out blood with the use of a fluid simulation. This software has been distributed to an international consortium with over 15 institutes.

The virtual simulation is written in C++, and heavily utilizes GPU technology to provide a realistic experience. The simulation relies on multithreading with specifically timed data transfers to utilize both CPU and GPU. The overlying simulation framework utilizes a JavaScript scripting engine to provide the user with a custom interface for site specific interactions.

The latest version features the ability to load patient-specific datasets for surgical rehearsal.

Interactive Virtual Environments

Combining consumer VR headsets with motion tracking cameras, IVE is a sophisticated environment for interacting and exploring different types of data without being limited by a tether.

The IVE system uses OptiTrack cameras for tracking, an Oculus Rift for visuals, connected to laptop-backpack for computation. Different environments were developed using Unity but these lacked the realism, then implemented these rooms using Unreal Engine.

Nasal simulation

Olfactory surgical simulation to test surgical intervention for air flow through nasal cavity. This tool loads patient-specific datasets and allows for removal within the sinuses. Currently, this information is exported to be used for air flow analysis using an offline CFD simulation.

GooFit

GooFit is a high-energy physics application where I improved GPU performance by over 50% for the latest GPU technology. These performance improvements relied on improving the memory access patterns. In addition, I was able to explore the feasibility of using embedding kernel calls within templated classes to allow for combining CUDA kernels into different arrangements at compile time.

VolEditor

VolEditor is a 3D painting tool, allowing for arbitrary rotation, translation, zooming of 3D volumetric datasets. A simple point-click interface allows a user to segment regions of the dataset with different segment values.
